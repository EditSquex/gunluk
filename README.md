# Kişisel Günlük

İnternette genel olarak güzel, sade ve gerçekten içe sinen bir günlük sitesi bulamadığım için; bi yerden sonra da artık bu düşünceleri ve bilgileri kafamda tutamadığım için böyle bir site yapmak istedim. Verilerinizi dış sunucularda saklamayan, tarayıcınızın kendi yerel hafızasında (LocalStorage) tutan ve anılarınızı hiçbir zaman kaybetmeyeceğiniz minimalist bir günlük platformu.

---

## Kurulum

Projeyi kendi yerel makinenizde çalıştırmak için:

```bash
# Repoyu klonlayın
git clone https://github.com/EditSquex/gunluk.git

# Proje dizinine girin
cd gunluk

# Bağımlılıkları yükleyin
npm install

# Geliştirici sunucusunu başlatın
npm run dev
```
