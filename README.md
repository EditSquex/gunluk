# 📜 Günlük — Kişisel Gotik Web Günlüğü

İnternette genel olarak güzel, sade ve gerçekten içe sinen bir günlük sitesi bulamadığım için; bi yerden sonra da artık bu düşünceleri ve bilgileri kafamda tutamadığım için böyle bir site yapmak istedim. Verilerinizi dış sunucularda saklamayan, tarayıcınızın kendi yerel hafızasında (LocalStorage) tutan ve anılarınızı hiçbir zaman kaybetmeyeceğiniz minimalist bir günlük platformu.

---

## 🖤 Öne Çıkan Özellikler

- **🔒 %100 Yerel Güvenlik:** Tüm verileriniz tamamen sizin cihazınızda saklanır. Hiçbir sunucuya veri gönderilmez.
- **🥀 Atmosferik Ruh Halleri:** Her ruh haline özel görsel parçacık efektleri (Sağanak Yağmur, Solgun Gül Yaprakları, Ateş Kıvılcımları, Kahve Buharı, Yıldız Yağmuru).
- **✒️ Özel Gotik Zengin Metin Editörü:** 
  - Özel font ailesi ve exact `px` punto boyut ayarları.
  - Kalın, italik, altı çizili, üstü çizili, alıntı ve renkli metin biçimlendirmesi.
  - Emoji paleti desteği.
- **✨ Çift Kürsör Deneyimi:** Sitede gezinirken Gotik Hilal (`🌙`), yazı yazma ekranına geçtiğinizde ise animasyonlu Gotik Sarı Kalem kürsörü.
- **📖 Kitap Formatında Okuma Tuvalı:** Anılarınızı tıpkı bir roman okur gibi ferah ve zengin bir pencerede inceleme imkanı.
- **📅 Takvim & Geçmiş Geçmişi:** Geçmiş günlere hızlıca ulaşın, anılarınızı tarihe göre filtreleyin ve istatistiklerinizi inceleyin.
- **💾 Veri Aktarımı (Yedekleme):** Günlüklerinizi dilediğiniz zaman `JSON` veya `TXT` formatında bilgisayarınıza indirebilirsiniz.

---

## 🛠️ Teknolojiler

- **Core:** React (Vite)
- **Styling:** Tailwind CSS (Gotik Minimalist Siyah Tema)
- **Icons & Visuals:** Lucide React, Canvas Confetti
- **Storage:** Web LocalStorage API

---

## 🚀 Kurulum ve Çalıştırma

Projeyi kendi yerel makinenizde çalıştırmak için:

```bash
# Repoyu klonlayın
git clone https://github.com/EditSquex/gunluk.git

# Proje dizinine girin
cd gunluk

# Bağımlılıkları yükleyin
npm install

# Geliştirici sunucusunu başlatın
npm run dev
```
